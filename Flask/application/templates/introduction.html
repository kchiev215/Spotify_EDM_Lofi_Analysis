About Me

Kristina “Thina” Chiev, Doctorate in Physical Therapy from USciences. Former outpatient and pediatric Physical Therapist. For her passion project, she collaborated with a member of the Java track to develop a rock climbing recommendation app on Spring Boot. The application recommends a boulder route based off difficulty of the route and location chosen. Her primary role included the implementation of ETL where she created web scraping tools to extract information, transformed the data into clean and easy to read format, and loaded the data to a SQL database to be used on Spring Boot with the usage of Jupyter Notebook. Familiarity with the process of ETL, Python and SQL language, the usage of tools and databases including Jupyter Notebook, NumPy, pandas, mySQL SQLite, and Agile methodologies.
